VIRTDISK.DLL EXPLANATION:
=========================

What is virtdisk.dll:
- Windows system library for Virtual Hard Disk operations
- Part of Windows since Windows 7/Server 2008 R2
- Provides low-level VHD/VHDX manipulation capabilities
- Used internally by Windows tools like Disk Management, PowerShell, Hyper-V Manager

Where virtdisk.dll is used in our implementation:

1. VHD CREATION (CreateVirtualDisk):
   - Direct file creation without WMI overhead
   - Support for Dynamic and Fixed VHD types
   - Support for both VHD and VHDX formats
   - Configurable block size and sector size

2. VHD OPENING (OpenVirtualDisk):
   - Opens existing VHD/VHDX files for manipulation
   - Provides handle for subsequent operations
   - Supports read-only and read-write access modes
   - Handles parent-child relationships for differencing disks

3. VHD ATTACHMENT (AttachVirtualDisk):
   - Mounts VHD as a disk drive to the host system
   - Makes VHD contents accessible as a regular disk
   - Supports permanent and temporary attachment modes
   - Can assign or suppress drive letter assignment

4. VHD DETACHMENT (DetachVirtualDisk):
   - Unmounts VHD from the host system
   - Cleanly releases all handles and resources
   - Ensures data integrity during detachment

5. VHD RESIZING (ResizeVirtualDisk):
   - Expands or shrinks VHD files
   - Supports both online and offline resizing
   - Handles dynamic and fixed VHD types differently
   - Includes safety checks for data integrity

6. VHD INFORMATION (GetVirtualDiskInformation):
   - Retrieves metadata about VHD files
   - Size information, parent relationships, format details
   - Fragmentation status and alignment information
   - Virtual disk identifiers and timestamps
